建立「Virtual Environments」，安裝「MkDocs」

建立專案資料夾

建立專案資料夾

$ mkdir note

切換到專案資料夾

$ cd note

建立「Virtual Environments」

執行 (注意這裡是pyvenv，中間有個v,非剛剛的pyenv)

$ pyvenv .env

會產生一個資料夾「.env」

確認目前執行的pip所在位置

執行

$ pip show pip | grep Location

顯示

Location: /home/user/.pyenv/versions/3.4.4/lib/python3.4/site-packages

進入「Virtual Environments」

執行

$ source .env/bin/activate

確認目前執行的pip所在位置

執行

$ pip show pip | grep Location

顯示

Location: /home/user/Project/note/.env/lib/python3.4/site-packages

安裝「mkdocs」

執行

$ pip install mkdocs

觀看「mkdocs」安裝到那個路徑

執行

$ pip show mkdocs | grep Location

顯示

Location: /home/user/Project/note/.env/lib/python3.4/site-packages

退出「Virtual Environments」

執行

$ deactivate

確認目前執行的pip所在位置

執行

$ pip show pip | grep Location

顯示

Location: /home/user/.pyenv/versions/3.4.4/lib/python3.4/site-packages
