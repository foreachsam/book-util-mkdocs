使用「pyenv」安裝「Python」

要使用「MkDocs」必須要有「Python」的環境，

本書撰寫的過程，測試的系統是在「Xubuntu 14.04 64位元」下寫成的，內建是有「Python」，

而我會習慣使用「pyenv」來安裝不同的「Python」來做一些測試，所以已經有「Python」的系統，就可以直接跳下一頁。

安裝「pyenv」

可以到「github / pyenv」觀看安裝說明。

先安裝一些編譯python的相依套件

$ sudo apt-get install make build-essential libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm

clone 「pyenv」

$ git clone https://github.com/yyuu/pyenv.git ~/.pyenv

設定

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(pyenv init -)"' >> ~/.bashrc

安裝「Python」

查詢目前有哪些版本可以安裝

$ pyenv install -l
