如何設定

前面「建立專案」時，有看到一個檔「mkdocs.yml」，

「設定」就是「編輯mkdocs.yml這個檔」。

觀看 「mkdocs.yml」內容

執行

$ cat mkdocs.yml

顯示

site_name: My Docs

觀看 「mkdocs.yml」內容

更改如下

site_name: 'MkDocs 學習筆記'

這時候就會看到網頁的標題改為「MkDocs 學習筆記」了
